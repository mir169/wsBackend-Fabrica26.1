Instruções rápidas (sem tecnologia)

1) Clique em "Login" e entre como admin (por ex. admin/admin123).
2) Visite guildas ou personagens para ver listagem (qualquer pessoa vê).

3) Se você for funcionário (staff), pode criar/editar/excluir.

4) Para usar a API diretamente (se quiser):

  1. Vá para /api/token/ e envie username/password para receber access (token).
  2. Use token no cabeçalho: Authorization: Bearer .
  3. URLs de leitura pública: /api/guilds/ (GET), /api/characters/ (GET).
  4. Para criar/editar/excluir API, você precisa ser is_staff.

Se estiver usando navegador: sempre use as páginas /guilds/ e /characters/, não a API, a não ser que souber usar token.
