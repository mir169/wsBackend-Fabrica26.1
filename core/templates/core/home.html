🚀 Como usar:

  1. Para CRUD Elden Ring: Faça login como admin (admin/admin123) para criar/editar dados de personagens, armas e chefes
  2. Para API Elden Ring: Navegue pelas seções sem necessidade de login
  3. API Status: Verde = API online, Amarelo = dados locais (offline)

Instruções rápidas (sem tecnologia)

1) Clique em "Login" e entre como admin (por ex. admin/admin123).
2) Visite as seções "Personagens ER DB", "Armas ER DB" ou "Chefes ER DB" para ver listagem (qualquer pessoa vê).

3) Se você for funcionário (staff), pode criar/editar/excluir.

4) Para usar a API diretamente (se quiser):

  1. Vá para /api/token/ e envie username/password para receber access (token).
  2. Use token no cabeçalho: Authorization: Bearer .
  3. URLs de leitura pública: /api/elden-characters/, /api/elden-weapons/, /api/elden-bosses/ (GET).
  4. Para criar/editar/excluir API, você precisa ser is_staff.

Se estiver usando navegador: sempre use as páginas /elden/characters/, /elden/weapons/, /elden/bosses/, não a API, a não ser que souber usar token.
